# MatBong

의성 지역 맛집을 지도에서 탐색하고 관리할 수 있는 **로컬 맛집 지도 서비스**입니다.

MatBong은 사용자가 주변 식당을 빠르게 찾고, 관리자가 지도 기반으로 매장 정보를 등록할 수 있도록 설계한 웹 서비스입니다.  
지도, 카테고리 필터, 검색, 정렬, 역할 기반 화면을 통해 실제 서비스에 가까운 맛집 탐색 흐름을 제공합니다.

## 프로젝트 개요

맛집 정보를 단순 리스트로만 보여주면 사용자는 매장의 위치, 거리감, 카테고리를 한 번에 파악하기 어렵습니다.

MatBong은 이 문제를 해결하기 위해 **지도 중심 탐색 화면**을 기본 구조로 잡았습니다.  
사용자는 지도에서 매장의 위치를 확인하고, 리스트에서는 영업 상태와 평점, 카테고리 정보를 함께 확인할 수 있습니다.

또한 관리자 계정은 매장 위치를 직접 입력하는 방식이 아니라 **지도에서 위치를 클릭한 뒤 매장 정보를 등록**할 수 있도록 구성했습니다.

## 핵심 목표

- 사용자가 의성 지역 맛집을 **지도 기반으로 빠르게 탐색**할 수 있게 만들기
- 카테고리, 검색어, 영업 상태, 정렬 조건으로 **필요한 매장만 좁혀보기**
- 사용자와 관리자의 화면을 분리해 **역할별 서비스 흐름** 만들기
- 관리자 페이지에서 지도 클릭 기반으로 **매장 등록 경험 개선하기**
- 추후 백엔드 API와 연결하기 쉽도록 **service, store, data 구조 분리하기**

## 주요 기능

### 사용자 기능

- 일반 사용자 로그인
- 지도 기반 맛집 위치 확인
- 맛집 리스트 조회
- 매장명, 설명, 태그 기반 검색
- 전체, 한식, 면류, 고기류, 분식, 카페/디저트, Best 카테고리 필터
- 영업중 매장만 보기
- 평점순, 최신순 정렬
- 리스트와 지도 마커 연동
- 카카오맵 길찾기 연결

### 관리자 기능

- 관리자 계정 로그인
- 관리자 전용 매장 관리 화면 접근
- 지도에서 위치 클릭 후 매장 등록 모달 열기
- 매장명, 설명, 카테고리, 평점, 운영 시간, 휴무일, 태그 입력
- 등록된 매장 리스트 확인
- 매장 삭제
- 사용자 화면과 동일한 지도 기반 관리 경험 제공

## 서비스 흐름

### 일반 사용자

```text
로그인
  -> 맛집 찾기 페이지 이동
  -> 지도에서 위치 확인
  -> 카테고리 / 검색 / 영업 상태 필터 적용
  -> 맛집 리스트 확인
  -> 원하는 매장 선택
  -> 카카오맵 길찾기 이동
```

### 관리자

```text
로그인
  -> 관리자 페이지 이동
  -> 지도에서 매장 위치 클릭
  -> 매장 정보 입력 모달 작성
  -> 매장 등록
  -> 등록된 매장 확인 및 삭제
```

## 구조 설계

### pages

페이지 단위 화면을 관리합니다.  
로그인, 사용자 맛집 탐색, 관리자 매장 관리 화면이 이 영역에 배치됩니다.

### components

화면에서 반복 사용되는 UI를 관리합니다.  
지도, 필터, 맛집 카드, 관리자 등록 모달처럼 역할이 명확한 UI를 컴포넌트로 분리했습니다.

### store

Zustand를 사용해 맛집 데이터, 로그인 사용자, 필터 상태, 선택된 매장 상태를 관리합니다.

### services

API 호출 구조를 담당합니다.  
현재는 localStorage 기반으로 동작하지만, `VITE_API_BASE_URL`을 설정하면 같은 호출 구조로 백엔드 API와 연결할 수 있도록 설계했습니다.

### data

서비스 초기 화면에 필요한 맛집 데이터, 카테고리, 태그 정보를 관리합니다.

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버 실행 후 아래 주소로 접속합니다.

```text
http://localhost:5173
```

## Kakao Map 설정

카카오맵이 정상적으로 렌더링되려면 Kakao Developers의 Web 플랫폼에 개발 도메인이 등록되어 있어야 합니다.

```text
http://localhost:5173
http://127.0.0.1:5173
```

도메인이 등록되어 있지 않으면 Kakao Map SDK가 차단되어 지도가 보이지 않을 수 있습니다.

## 프로젝트 방향

MatBong은 단순한 맛집 리스트 서비스가 아니라, **사용자가 지도에서 위치를 보고 바로 판단할 수 있는 지역 맛집 탐색 서비스**를 목표로 합니다.

프론트엔드는 실제 서비스처럼 사용자와 관리자의 동선을 분리하고, 백엔드는 추후 API 연결을 전제로 확장 가능한 구조를 갖추는 방향으로 설계했습니다.
